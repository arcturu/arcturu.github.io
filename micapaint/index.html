Micapaint

[English] [日本語]

A painting app for iPad that pursues the ultimate drawing experience.

Features

  • Smooth brush performance at 120fps: even 1000×1000 blending brushes run fluidly.
  • "Slime Brush" that fills regions without spilling outside broken lines.
  • A simple, workspace-first UI that maximizes your canvas.

Basic Functions

  • Pencil, G-pen, watercolor blending, blur
  • Lasso fill
  • Unlimited layers (as long as memory allows)
  • Layer folders
  • Clipping masks (yes, even for folders!)
  • Canvas flip for preview
  • Mini canvas view
  • Alpha lock
  • Filters (noise, hue/saturation/brightness)
  • Selection & transform
  • Image import
  • Standard blend modes
  • Undo / redo
  • Adjustable stabilizer strength
  • Barrel roll support
  • Customizable toolbar (button count & layout)
  • Timelapse recording
